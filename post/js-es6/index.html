ES6

  • let则实际上为JavaScript新增了块级作用域。用它所声明的变量，只在let命令所在的代码块内有效。

    var a = [];
for (let i = 0; i < 10; i++) {
a[i] = function () {
console.log(i);
};
}
a[6](); // 6

  • const用来声明常量。一旦声明，常量的值就不能改变。

    const monent = require('moment')

  • arrow function

    class Animal {
constructor(){
    this.type = 'animal'
}
says(say){
    setTimeout( () => {
        console.log(this.type + ' says ' + say)
    }, 1000)
}
}
var animal = new Animal()
animal.says('hi')  //animal says hi
