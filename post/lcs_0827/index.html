51. N-Queens

  • dfs: each recursion check with previous rows
  • quick check: using a boolean array to record if current location is conflicts with elements in col, \ and /
  • bit mask:
    • set i to 1: a |= (1 << i)
    • set i to 0: a &= ~(1 << i)
    • inverse i: a ^= (1 << i)
    • get i: (a >> i) & 1
    • find last 1: a & -a
